# [Chat.goomba](https://chat.goomba.org) API

API for chat.goomba

## Environment

Java 17  
Mysql 8.0  
Gradle

Spring boot 3.3.2  
Spring JPA  
Spring Data JPA  
Querydsl

jwt  
websocket  
lombok  
sentry  
thumbnailator  
imageio

## Deployment

```
git clone https://github.com/GOOMBANGEE/chat-backend.git

cd chat-backend
nano .env

./gradlew build # Linux/mac
./gradlew.bat build # Windows

java -jar ./build/libs/chat-0.0.1.jar
```
